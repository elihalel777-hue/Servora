# Servora

Servora is an AI-powered maintenance operations platform for property managers.

## Current Features

- Homepage
- Login page
- Create account page
- Manager dashboard
- Properties
- Buildings
- Units
- Maintenance requests
- AI request analysis
- Work orders
- Technician portal
- Backend starter files

## Backend

The backend uses:

- Node.js
- Express
- CORS

## Start Command

```bash
npm start
